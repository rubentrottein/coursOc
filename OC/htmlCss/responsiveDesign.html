Responsive Design et Media Queries

On peut utiliser l'attribut "media" dans la balise link afin de specifier le fichier qui définira les regles à employer pour chaque type d'écrans auxquels on souhaite s'adapter

On peut créer des media queries, qui permettent de placer du code différent selon le viewport dans le fichier CSS (cf le css de cette page

En dehors du CSS, on peut spécifier dans des balises telles que < meta name="viewport" content="width=device-width" />
